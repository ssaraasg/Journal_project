This is a mood tracker and a digital diary notebook. The user can track and journal any details of any date she wants and also read the saved journals. The project default journal date is today but by choosing the journal list, she can select a desired date and start journaling at that date. 
Based on the mood user selected, the journaling page will change: 
Happy : pink 
Sad : Blue 
Mad : Brown 
Confusing: Gray 
If user want to see a specific date , again based on the of that day, the page color would be different. 
 
Instruction:
Install the tkcalender by run this code to successfully load the program: 
Pip install tkcalender
